This repository documents the learning process, laboratory exercises, and projects related to **Microcontrollers and Embedded Systems** during the **6th semester** of study.

The goal of this repository is to build a strong understanding of **low-level hardware control, embedded programming, and real-world microcontroller applications**.

# Learning Objectives

This repository focuses on the following key objectives:

* Understand **microcontroller architecture**
* Implement **low-level hardware control**
* Develop **embedded applications using C**
* Learn **peripheral communication protocols**
* Interface **sensors and actuators**
* Build small **embedded automation systems**

# Example Learning Workflow

```
Theory → Hardware Design → Firmware Development → Testing → Debugging → Optimization
```

Typical workflow in this repository:

1. Study microcontroller theory
2. Implement firmware in Embedded C
3. Interface with external hardware
4. Test behavior using serial debugging
5. Optimize performance and reliability
